### cocoron 
感情と日記を記録するスマホ向け心のWEBログアプリ

### コンセプト
日々モヤモヤと感情を巡らせるあなたに。
「文章 + 気分アイコン」で記録し、当時を振り返りやすくすることを目的とした日記アプリ。
その日のメモと mood（気分）を記録し、カレンダーから過去の日付をタップすると当時の記録を振り返ることができます。

### 技術
- HTML / CSS / Vanilla JavaScript
- localStorage（データ永続化）
- Date API
※ フレームワーク（React / Vue 等）は使用していません。

### 主な機能
- 日付ごとのメモ管理
- 1日3件まで"気持ち"を記録
- mood（☀️ / ☁️ / 🌀）の登録
- カレンダー連動表示（月切り替え対応）
- カレンダー上にmoodアイコンの表示
- 日付タップで当日のメモを表示
- localStorage を使った永続化
- スマホUIを意識したデザイン

### こだわった点
1. Vanilla JS での状態管理
React などのフレームワークを使わず、以下の状態を 自前で管理しています。
- 選択中の日付（selectedDateKey）
- 日付ごとのログと mood（cocoroData）
- カレンダー表示中の月（currentMonth）
UI 操作ごとに「状態 → 描画」の流れを意識して実装しました。

2. データ構造の設計
cocoroData = {
  "2026-01-20": {
    logs: ["memo1", "memo2"],
    mood: "sun"
  }
}
- 日付をキーにしたオブジェクト構造
- メモは配列で管理
- mood は文字列で管理
後から機能追加しやすい構成を意識しています。
